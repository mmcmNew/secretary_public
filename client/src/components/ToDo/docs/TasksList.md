# TasksList Component

## Описание

`TasksList` - это компонент, отвечающий за отображение списка задач. Он предназначен для максимальной переиспользуемости и "тупости", т.е. не содержит сложной бизнес-логики, а лишь отображает данные и передает события родительским компонентам.

## Взаимодействие компонентов

`TasksList` использует вспомогательные компоненты:
- `TaskItem` - для отображения отдельной задачи (мемоизированный компонент)

Также он взаимодействует с хуками:
- `useTasksLogic` - для управления логикой задач
- `useTasks` - для работы с API задач

## Задачи компонента

1. Отображение списка задач (активные и завершенные)
2. Обработка пользовательских действий (клик, контекстное меню, drag & drop)
3. Передача событий и данных родительским компонентам
4. Управление состоянием UI (открытие меню, отображение подзадач и т.д.)

## Зоны ответственности

- Отображение данных задач
- Обработка пользовательских событий
- Управление состоянием UI
- Передача данных и событий через props

## Props

| Название | Тип | Обязательный | Описание |
|----------|-----|--------------|----------|
| `containerId` | `string` или `number` | Нет | ID контейнера для drag & drop |
| `listsList` | `array` | Нет (по умолчанию `[]`) | Список пользовательских списков |
| `projects` | `array` | Нет (по умолчанию `[]`) | Список проектов |
| `selectedList` | `object` | Нет | Выбранный список |
| `isNeedContextMenu` | `boolean` | Нет (по умолчанию `false`) | Флаг необходимости отображения контекстного меню |
| `setSelectedTaskId` | `function` | Нет (по умолчанию `null`) | Функция для установки выбранной задачи |
| `linkTaskList` | `function` | Нет (по умолчанию `null`) | Функция для связывания задачи со списком |
| `deleteFromChildes` | `function` | Нет (по умолчанию `null`) | Функция для удаления из дочерних элементов. Удаление из списка или удаление подзадачи из списка подзадач задачи |
| `additionalButton` | `object` | Нет (по умолчанию `null`) | Дополнительная кнопка для задачи |
| `additionalButtonClick` | `function` | Нет (по умолчанию `null`) | Обработчик клика по дополнительной кнопке |
| `tasks` | `array` | Нет (по умолчанию `[]`) | Список задач |
| `selectedTaskId` | `number` | Нет (по умолчанию `null`) | ID выбранной задачи |
| `updateTask` | `function` | Нет (по умолчанию `null`) | Функция для обновления задачи |
| `changeTaskStatus` | `function` | Нет (по умолчанию `null`) | Функция для изменения статуса задачи |
| `onSuccess` | `function` | Нет (по умолчанию `null`) | Функция для вывода пользователю уведомления об успехе через общий контекст уведомлений |
| `onError` | `function` | Нет (по умолчанию `null`) | Функция для вывода пользователю уведомления об ошибке через общий контекст уведомлений |

## Структура компонентов

Компонент `TasksList` использует компонент `TaskItem` для отображения отдельных задач. `TaskItem` был вынесен в отдельный файл `TaskItem.jsx` для улучшения читаемости и поддержки кода.
