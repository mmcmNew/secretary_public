# TasksContext

## Описание

`TasksContext` - это контекст React, который предоставляет доступ к данным и функциям для работы с задачами, списками, календарем и другими связанными сущностями в ToDo-приложении. Он предназначен для централизованного хранения состояния и предоставления общего API для всех компонентов ToDo.

## Взаимодействие компонентов

`TasksContext` использует хуки:
- `useTasks` - для работы с задачами
- `useLists` - для работы со списками
- `useMyDay` - для работы с задачами "Мой день"
- `useCalendar` - для работы с календарем
- `useTaskTypes` - для работы с типами задач

Также он использует `react-query` для управления состоянием данных и мутациями.

## Задачи контекста

1. Централизованное хранение состояния ToDo-приложения
2. Предоставление единого API для работы с данными и функциями
3. Управление состоянием выбранных элементов
4. Обработка мутаций данных (добавление, обновление, удаление задач и списков)

## Зоны ответственности

- Хранение и предоставление данных о задачах, списках, календаре
- Управление состоянием выбранных элементов
- Обработка пользовательских действий через мутации
- Синхронизация данных между компонентами

## Значения контекста

| Название | Тип | Описание |
|----------|-----|----------|
| `tasks` | `object` | Объект с данными о задачах |
| `myDayTasks` | `object` | Объект с данными о задачах "Мой день" |
| `myDayList` | `object` | Объект с данными о списке "Мой день" |
| `lists` | `object` | Объект с данными о списках задач, проектах и группах |
| `calendarEvents` | `array` | Массив событий календаря |
| `calendarRange` | `object` | Объект с диапазоном дат календаря |
| `selectedTaskId` | `number` | ID выбранной задачи |
| `taskFields` | `object` | Объект с описанием полей задачи |
| `selectedList` | `object` | Выбранный список |
| `selectedListId` | `string` или `number` | ID выбранного списка |
| `fetchTasks` | `function` | Функция для получения задач |
| `fetchLists` | `function` | Функция для получения списков |
| `fetchMyDayTasks` | `function` | Функция для получения задач "Мой день" |
| `fetchCalendarEvents` | `function` | Функция для получения событий календаря |
| `addTask` | `function` | Функция для добавления задачи |
| `updateTask` | `function` | Функция для обновления задачи |
| `deleteTask` | `function` | Функция для удаления задачи |
| `changeTaskStatus` | `function` | Функция для изменения статуса задачи |
| `addList` | `function` | Функция для добавления списка |
| `updateList` | `function` | Функция для обновления списка |
| `deleteList` | `function` | Функция для удаления списка |
| `getTaskTypes` | `function` | Функция для получения типов задач |
| `addTaskType` | `function` | Функция для добавления типа задачи |
| `getTaskTypeGroups` | `function` | Функция для получения групп типов задач |
| `setSelectedTaskId` | `function` | Функция для установки выбранной задачи |
| `setSelectedListId` | `function` | Функция для установки выбранного списка |
| `setCalendarRange` | `function` | Функция для установки диапазона дат календаря |
| `linkListGroup` | `function` | Функция для связывания списков/групп |
| `deleteFromChildes` | `function` | Функция для удаления из дочерних элементов |
| `linkTaskList` | `function` | Функция для связывания задачи со списком |
| `getSubtasksByParentId` | `function` | Функция для получения подзадач по ID родительской задачи |
